# 02-진행사항

프로젝트의 현재 완료 상태와 앞으로의 진행 계획을 정리한 문서입니다.

---

## 현재 완료된 기능

| 구분 | 내용 | 비고 |
|------|------|------|
| **프로젝트 구성** | Spring Boot 3.5.x, Java 17, Gradle | - |
| **MongoDB 연동** | spring-boot-starter-data-mongodb, MongoTemplate 사용 | 컬렉션: user, user2, memo |
| **ODM 도큐먼트** | Users, User2, Memo 정의 | @Document, @Id 등 적용 |
| **Spring Security** | form 로그인, 이메일 파라미터, BCrypt, 역할(USER/ADMIN) | CustomAuthenticationEntryPoint 적용 |
| **회원** | 회원가입(joinForm/joinUser), 로그인/로그아웃, UserDetailsService(MongoDB 조회) | - |
| **권한** | `/admin` ADMIN 전용, 그 외 인증 필요, 정적 리소스·메인·로그인·회원가입 permitAll | - |
| **메모 CRUD** | 등록(일반/이미지 첨부), 수정, 삭제, 목록 조회 | - |
| **검색** | 메모 제목·메시지·_id 조건 검색(SearchDB) | - |
| **이미지·파일** | GridFS 업로드/다운로드, 썸네일 생성(이미지), 파일명 목록·삭제 | 동영상은 썸네일 제외 등 확장자 분기 |
| **뷰** | Thymeleaf + layout, main, admin, login, join, updateForm 등 | - |
| **기본 관리자** | 기동 시 admin/admin1234 자동 생성 (DefaultAdminInitializer) | 없을 때만 1회 생성 |
| **로그인 리다이렉트** | 로그인 성공 시 중간 경유(/login/redirect) 후 목적지 이동 | 새로고침 없이 정상 진입 |
